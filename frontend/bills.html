Help
How to pay a bill?
Bill type select pannunga, provider name, customer ID, amount and payment method enter pannunga. Password / PIN kuduthu Pay Bill press pannunga.
Receipt save aguma?
Yes. Neenga pay panna odane receipt history la save agum. Top la iruka ⋯ button use panni History open pannalaam.
Wrong password vandha?
Session la store panna password match aaganum. Wrong password / PIN kudutha payment proceed aagathu.
Balance deduct aguma?
Yes. Payment success aana amount wallet balance la irundhu deduct aagum. Insufficient balance na transaction block aagum.
